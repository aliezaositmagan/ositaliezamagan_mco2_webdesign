MY REFLECTION

In Exercise 3, I learned how to use HTML to add lists, images, videos, and audio to a web page. At first, I felt a bit confused because each element has different tags and attributes. But as I practiced, I started to understand how they work together to make a webpage more complete and interesting. I enjoyed working with lists because they helped me organize information clearly. Adding an image was also fun because it made the page look more attractive. The video and audio parts were a bit challenging at first, especially when linking the correct file paths. But once I figured it out, it became easier, and I felt happy seeing the media play on the page.
