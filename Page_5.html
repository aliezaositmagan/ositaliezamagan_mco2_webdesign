MY REFLECTION

In this activity, I learned how to create a simple Personal Schedule Page using HTML. At first, I felt a bit unsure about how to arrange the elements properly, but as I continued working, I became more comfortable with the tags and layout. I learned how to use headings, lists, tables, and other basic HTML elements to organize my daily schedule clearly. This exercise helped me understand how important structure is when creating a webpage. I also realized that even small details, like spacing and correct tag usage, can make the page look cleaner and easier to read. While doing the activity, I enjoyed seeing my work slowly come together on the browser, and it made me more confident in using HTML.
