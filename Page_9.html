MY REFLECTION

In Exercise 9, I was able to combine and apply all the CSS properties we learned from our previous exercises to create a complete and well-structured webpage layout. This activity helped me see how each property—such as margins, padding, borders, colors, fonts, background styling, and layout techniques like flexbox and positioning—works together to improve both the design and organization of a webpage. By putting everything into one project, I better understood how important proper spacing, alignment, and visual styling are in making a webpage look clean and easy to navigate. Overall, this exercise strengthened my confidence in using CSS and showed me how these tools can transform simple HTML into a polished and visually appealing layout.
